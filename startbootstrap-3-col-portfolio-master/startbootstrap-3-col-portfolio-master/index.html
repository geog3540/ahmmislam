Trends of changes in tree canopy cover and impervious surface An explantion of recent changes

Final Project Proposal

The goal of this project is to observe the relationship between the tree cover changes of Iowa with the amount of developed impervious surface from 2001-2019. My hypothesis will be, there is going to be a negative change in tree cover while the amount of developed impervious surface is on the rise in CA.

Output 01: Bivariate Analysis

The aim of bivariate analysis between tree canopy cover change and impervious surface change is to identify the top counties which are observing a high rate of canopy ans surface change in recent times.

Output 02: Chart Analysis

The aim of this chart analysis is to check the county level specific changes.

Output 03: Choropleth Analysis

The aim of this small choropleth maps is to correlate canopy cover and surface changes with GDP and population changes in California counties.
